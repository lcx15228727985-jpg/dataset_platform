若在 Cursor 或 PowerShell 中无法启动可视化平台，请按下面方式启动：

1. 打开资源管理器，进入项目根目录：
   D:\PyCharmProject\gpr_prj\IROS

2. 双击运行：
   dataset_viewer\run.bat

3. 等待命令行窗口出现 “You can now view your Streamlit app in your browser” 后，
   浏览器会自动打开 http://localhost:8501 ；若未自动打开，请手动在浏览器输入该地址。

或：按 Win+R，输入 cmd 回车，在 CMD 中执行：
   set SSLKEYLOGFILE=
   cd /d D:\PyCharmProject\gpr_prj\IROS
   python -m streamlit run dataset_viewer/app.py
